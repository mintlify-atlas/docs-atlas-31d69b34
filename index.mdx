## Overview

SMyEG (Sistema de Monitoreo y Evaluación Geoespacial) is a comprehensive full-stack geospatial monitoring and evaluation platform designed for forest patrimony management. Built with Next.js, React, and PostgreSQL, it provides powerful tools for managing forest assets, tracking biological resources, and analyzing land use across hierarchical organizational structures.

## Core Capabilities

- **Authentication & Authorization**: NextAuth v5 with credential provider and role-based permissions
- **Shapefile Import**: Validate and process geospatial data with automatic hierarchy linking
- **Land Accounting**: Calculate forest areas and track land use changes over time
- **Measurement Campaigns**: Create and manage biological asset measurement campaigns
- **Dashboard Analytics**: Real-time metrics and visualizations for decision-making
- **Multi-tenant Architecture**: Secure organization isolation with comprehensive access control
